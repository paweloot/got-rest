# GOT Mobile - REST API
REST API used by [GOT Mobile](https://github.com/paweloot/got-mobile "GOT Mobile Repository") to exchange data with MySQL database.

## Built using:
  * Java 8
  * Hibernate
  * Spring Boot
  * Spring Data JPA
